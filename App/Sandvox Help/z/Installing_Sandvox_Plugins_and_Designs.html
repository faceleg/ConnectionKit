Installing Sandvox Plugins and Designs

How a Sandvox plugin or design appears in the Finder.

Sandvox has a flexible plugin architecture. This means that it is possible to extend Sandvox's capabilities by installing a component created by a third party.

Sandvox allows you to install additional:

  • Designs
  • Page types
  • Pagelet types
  • Data sources (allows you to drag and drop additional kinds of media into Sandvox)

To install a new plugin or design:

  1. Double-click the plugin or design in the Finder.
  2. Quit and relaunch Sandvox.
  3. You can then delete the original plugin file if you wish, since Sandvox will have copied it as part of the installation.

Using this method, a plugin or design is placed in the "Library → Application Support → Sandvox" folder in your Home Directory.

You can actually place a plugin or design in any of these folders to install it:

  • ~/Library/Application Support/Sandvox - installed for just you
  • /Library/Application Support/Sandvox" - installed for all users of the computer
  • /Network/Library/Application Support/Sandvox - for use in a special network set-up

in a location higher up the list take priority over those lower down. The application package itself is checked last.

To uninstall a plug-in or design:

  1. Open the "Application Support → Sandvox" folder containing the plugin or design in the Finder.
  2. Remove the plugin file from that folder.
