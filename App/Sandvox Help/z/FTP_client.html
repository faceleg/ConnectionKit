FTP client

An FTP client allows you to connect to a host and manually manipulate the files stored there. This could be useful for:

  • Deleting a published site
  • Uploading an exported site
  • Sending Karelia a large file as part of a support request

You can generally connect using either the FTP, SFTP, or WebDAV protocols - most clients support all three. A number of FTP clients are available for OS X. Some of the major offerings are:

  • Cyberduck (freeware)
  • Transmit
  • Interarchy
  • Yummy FTP
  • Fetch
  • Captain FTP
