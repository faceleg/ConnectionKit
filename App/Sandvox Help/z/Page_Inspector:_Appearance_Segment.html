Page Inspector: Appearance Segment

Layout

Page Title

This displays the page's title on the page. Uncheck the box if you do not want the page title to show.

If hidden, the title can still be edited by double-clicking on the page in the Site Outline

Date Created, Modified

Check the box to display the page's date and time of creation, or modification. To choose either the creation date or modification date, click the small arrows to the right, that will bring up a popup menu.

Sidebar

Hide or show the sidebar on this page by checking or unchecking the sidebar box. You can also control precisely which objects appear in the sidebar.

Comments

Check the box to allow visitors to leave comments on the page. If not already done so, the commenting system must be configured using the "Setup" button.

Thumbnail

To select a custom image, select "Image" from the popup, and navigate to the image in the Finder window.
