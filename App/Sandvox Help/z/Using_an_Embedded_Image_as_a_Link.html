Using an Embedded Image as a Link

Embedded images can be used as links just like normal text.

To turn an embedded image into a link:

  1. Click the image once to select it.
  2. Click the "Create Link" item in the toolbar.
  3. Drag the target icon to a page in the Site Outline or enter the address of another webpage.
  4. Close the Link Destination Panel by clicking outside it.
