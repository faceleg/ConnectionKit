RSS Feed

Many websites provide an RSS feed allowing people to subscribe to it. An RSS Feed object takes such a feed and displays the headlines inside a page on your site. Visitors can click on an item in the page to read the full article.

Ways to add an RSS Feed to a site:

  • Use the "Objects" items in the toolbar.
  • Choose "More" from the Insert menu.

Note: Although you can drag the URL of the RSS feed from a web browser or another application to the Site Outline, doing so will not create an RSS Feed; rather it will create an External Page, and you will not have the same ability to control the way the RSS Feed appears on your site. So although this is a possible way to incorporate an RSS Feed, it's not generally recommended.

Editing

To edit an RSS Feed object:

  1. Click the page in the Site Outline to select it.
  2. Click the RSS Feed object to select it.
  3. Open the Object Inspector. Here you can edit settings specific to the RSS feed as shown below.

Full URL of Feed

The full URL of the RSS feed. For information on how to find the URL of a feed, please see the "RSS" article.

Entries

Limits the number of items displayed in the object. If you're incorporating a busy feed and don't want it to take up a lot of space in your site, set a limit here to control how long the object will be. If you enter "0" there will be no limit.

Summaries

Entering a number here will add summaries under each headline, up to the number of characters specified. Leaving this field blank or entering a 0 will eliminate the summaries.

Open in new window

Checking this box will open a new browser window for each link in the RSS feed.

Implementation Details

RSS Feed objects use Karelia Services to handle RSS feeds. The server takes a feed and transforms it into HTML code that is displayed on your site. Karelia uses MagpieRSS to power this service.

To ease the load on the sites whose RSS feed is to be displayed, each RSS feed is cached and updated once an hour.
