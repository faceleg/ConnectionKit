General Pagelet Attributes

Introduction

To edit a pagelet, you use the Selection Inspector. This contains settings that are applicable to all types of pagelet and those that apply only to the current pagelet's type.

This article deals with only those attributes applicable to all types of pagelet. For more information on the other, pagelet type-specific attributes, please see the articles on each type of pagelet.

The Attributes

Show border

If the site's design supports borders, checking this box will display a border around the pagelet.

Location

Place the pagelet in the Sidebar or within the page as a callout. Note that some page types do not allow callouts.

If you select "Sidebar", two further options become available:

Prefers bottom
If the pagelet is set to appear on any child pages then Sandvox will generally place it at the top of the Sidebar. With this option checked, the pagelet will instead appear at the bottom of the Sidebar.
Show on child pages
If the pagelet is placed in the Sidebar of a Collection, then checking this option will make it appear in the Sidebar of every page in the Collection.

Pagelet Position

If you have multiple pagelets grouped together, you may wish to change their ordering. Click the up or down arrows to do so. Note that if a pagelet is set to "prefers bottom," you will not be able to move it from the bottom of the Sidebar using these controls.
