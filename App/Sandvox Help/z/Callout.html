Callout

A callout is a pagelet set to appear within the main content of a page rather than the Sidebar.

A pagelet set as a callout. Notice how the main text of the page flows around the callout.

Any pagelet can be used as a callout, but some types of pages do not allow callouts.

Callouts appear on the left or right of a page, depending on the site's design.

To set a pagelet as a callout:

  1. Click the pagelet to select it.
  2. Open the Selection Inspector.
  3. Set the "Location" of the pagelet to "Callout."
