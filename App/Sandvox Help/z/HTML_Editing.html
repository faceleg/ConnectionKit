HTML Editing

Due to the way Sandvox works, you cannot edit the HTML code of an arbitrary page. However, with Sandvox Pro, you can:

  • Create Raw HTML pages and pagelets
  • Edit the HTML code of any text area.
  • Use Code Injection

For more information, please see the "Sandvox Pro" article.
