Sandvox Menu Guide

  • Commands that require Sandvox Pro are indicated like this -

Sandvox

About Sandvox

Displays the about box for Sandvox. Useful for checking which version of Sandvox you are running.

Check for Updates

Manually checks to see whether updates to Sandvox are available.

Discover Plug-ins

Opens the Discover Plug-ins Window.

Buy/Register Sandvox…

Displays the window used to buy or register Sandvox. If you have already bought Sandvox, then this can be used to upgrade to Sandvox Pro. For more information, please see "Purchasing Sandvox."

Visit Support Forum

Links directly to Karelia's online support forum for Sandvox.

Send Feedback

Opens the built-in feedback reporter.

Join Karelia Email List

Allows you to join Karelia's spam-free email list to receive occasional news, updates, or discounts. You can unsubscribe at any time.

Preferences…

Displays the Sandvox preferences window.

Services

This menu is a big list of all the services other applications on your computer can provide.

Hide Sandvox

Hides (but does not close) Sandvox.

Hide Others

Hides (but does not close) all applications other than Sandvox. Can be undone by selecting "Show All."

Show All

Shows all applications, even those hidden.

Quit Sandvox

Quits Sandvox.

File

New Site

Creates a new site from scratch.

Open…

Opens an existing site.

Open Recent

A list of sites you have recently worked on.

Close

Closes the current site.

Save Snapshot

Saves a snapshot of the current site document.

Save As…

Saves the current site document to a new location and begins editing it.

Save a Copy As…

Like the "Save As…" command, saves a copy of the current site document, but continues to work on the existing document.

Revert to Snapshot…

Reverts to the last snapshot of the document.

Edit

Undo and Redo

Undo and redo changes to the site.

Cut & Copy

Cuts or copies the current selection to the clipboard. See "Clipboard Menu Commands" for more information.

Paste Commands

Sandvox has three paste commands:

  • Paste
  • Paste Link
  • Paste and Match Style
  • Paste HTML into Text


Please see the "Clipboard Menu Commands" article for more information

Delete

Deletes the selected object. This command cannot be used to delete pages or pagelets. Instead use the commands below.

Delete Page

Deletes the selected page. If the page is a Collection then the menu changes to "Delete Collection."

Delete Pagelet

Deletes the selected pagelet.

Duplicate

Creates a duplicate copy of the selected object.

Select and Deselect All

When editing text, this selects or deselects all of the text in the box.

Create/Edit Link…

Turns the selected text into a link or edits an existing link.

Edit Raw HTML…

Edits the raw HTML code of the selected text area.

Find

Standard commands for searching text.

Spelling

A menu of standard spelling commands. To disable or enable spell checking as you type, you must set the appropriate preference.

Special Characters…

Shows the character palette for easy insertion of unusual characters into your site.

Format

Show/Hide Fonts

Toggles the Fonts panel, which allows you to set the font of the selected text.

Bold, Italic, Underline, & Strikeout

Applies the chosen effect to the selected text. Note that depending on the location and font of the text, not all of these may be available.

Show/Hide Colors

Toggles the Color Picker which allows you to set the color of the selected text.

Copy & Paste Style

Copies and pastes just the style of the selected text. See "Clipboard Menu Commands" for more information.

Clear Style

Resets the formatting of the selected text to the default for the page and design. See "Text Editing" for more information.

Alignment

Sets the alignment of the selected text.

Baseline

You can raise or lower the text baseline to superscript or subscript. This is useful when placing references or simple formulae on a site.

View

Show/Hide Designs

Toggles the visibility of the Design Chooser. There is also a toolbar item to do the same.

Show/Hide Status Bar

Toggles the visibility of the Status Bar at the bottom of the window.

Show/Hide Editing Markers

Toggles the visibility of Editing Markers. Equivalent to the "Show/Hide Markers" toolbar item.

Show/Hide Site Outline

Toggles the visibility of the Site Outline.

Small Page Icons

If checked, makes the icons in the Site Outline smaller. Useful if you have a large site and limited screen space.

Text Size

Sandvox provides three commands for setting the text viewing size:

  • "Make Text Bigger,"
  • "Make Text Smaller,"
  • "Normal Text Size."

Please see the "Web View" article for more information.

Reload Web View

Reloads the page you are viewing just like you would with a web browser. This is useful for dealing with items that draw their content from external sources, such as External Pages.

Web View

Provides alternative views for the Web View and an HTML Validation menu command. For more information on the alternative views, please see the "Web View" article.

Show/Hide Toolbar

Toggles the visibility of the toolbar. This can also be accomplished by clicking the button in the very top right of the window.

Customize Toolbar…

Displays a sheet allowing you to customize the items in the toolbar to better suit you.

Site

Add Page

Adds a new page of your choice to the site. This is equivalent to the "Pages" item in the toolbar.

Add Pagelet

Adds a new pagelet of your choice to the current page. This is equivalent to the "Pagelets" item in the toolbar.

Add Collection

Adds a new collection of your choice to the site. This is equivalent to the "Collections" item in the toolbar.

Group as General Collection

Groups the selected pages into a new Collection. Equivalent to the "Group" item in the toolbar. If no suitable pages are selected, the command is disabled.

Site Code Injection and Page Code Injection

Opens the Code Injection window for the Home Page (Site Code Injection) or the page selected in the Site Outline (Page Code Injection).

Setup Host…

Opens the Host Setup Assistant, which is used to configure how your site is published. If you are not sure what to do, it is a good idea to read "Setting Up Your Host."

Publish Changes

Publishes the site, but only those parts that have changed since the last time you published. Generally quicker than publishing the entire site.

Before you can publish a site, you must set up your host.

Publish All

Publishes the entire site, regardless of what changes have been made since the last publish.

Before you can publish a site, you must set up your host.

Export…

Places a complete web-ready copy of the site in a location of your choosing. For more information on how or why you would do this, please see "Publishing Your Site."

Visit Published Site and Visit Published Page

Opens the published site in your web browser.

Submit to Site Directory…

Opens the a page in your web browser where you can submit your website to a directory of Sandvox-created websites. Doing so (among other tasks) will help you to gain exposure to your website.

Window

Show/Hide Inspector

Toggles the Inspector.

Show/Hide Media Browser

Toggles the Media Browser.

Minimize

Minimizes the document window to the Dock.

Zoom

The zoom button (the little green circle at the top left of a window) toggles the size of the document window between filling the screen and how it was before.

Bring All To Front

Places all Sandvox windows in front of anything else you may have open. Hold down Option to neatly arrange all Sandvox windows.

Window List

At the bottom of the Window menu is a list of all the windows Sandvox has open.

Help

Sandvox Help

Opens the Sandvox Help (what you're reading now!).

Video Introduction

Opens the Sandvox video introduction, standard or full size.

Release Notes

Opens the Sandvox release notes.

Acknowledgments

Opens the Sandvox Acknowledgments window. This lists all the contributions that were necessary to make Sandvox happen.

Publishing Transcript

Opens a window that will show diagnostic information for publishing.
