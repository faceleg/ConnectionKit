Preferences

Like most Mac applications, Sandvox has a Preferences Window which you open by choosing "Preferences…" from the Sandvox menu.

Application Preferences

Checking for Updates

Sandvox can check for new versions automatically or manually. Using the popup menu, you can choose among the following:

Manually
Checks only when manually using the Check for Updates menu item or the Check Now button in the Preferences window. Not recommended.
Release versions only
Automatically notifies you of any newly released versions of the application.
Include beta versions
Automatically notifies you of new releases, plus cutting-edge "beta" versions. Recommended for experienced users only.

When opening document

There are three choices available for what Sandvox will do when opening a document:

Take snapshot
Takes a snapshot of the document which you can revert to later.
Create backup
A duplicate of the site is created in the same folder. For more information, please see "Site backups."
Don't create backup
No special action is taken.

Reporting Problems

Sandvox can report problems to Karelia in order to help improve the program in the future. You can set this to function one of the following:

Ask each time
Ask you whether to send a report each time a problem is encountered.
Automatically
Send problem reports automatically without asking. This is recommended.
Never
You will not be asked to report problems to Karelia. Not recommended.

You can use the From field to specify the default email address attached to problem reports. This is helpful should Karelia need to contact you about your report.

General Preferences

Show "live" data feeds and pages

Whether to download content from the Internet while editing a site. See the "Live Data Feeds" article for more information.

Check spelling as you type

When this option is checked, Sandvox will check text for spelling errors as you type.

If you choose to disable this setting, then you can check your spelling at any time by using the "Spelling…" or "Check Spelling" commands from the "Edit → Spelling" menu.

Set creation date of pages from date of source material

You can create a new page by dragging an appropriate piece of data into the Site Outline. For example, dragging an image to the Site Outline results in a Photo page.

With this option checked, the creation date of the new page is set to be the creation date of the item you dragged (if it has one), rather than the current date.

Automatically read URLs from web browser

When you create an item that uses an external web address (a link or a YouTube page, for example), this option will automatically use the address of the current web page in your default browser. This feature is supported for Safari, Opera, Camino, Firefox, and OmniWeb.

Note that this feature can save you time and prevent a lot of errors, since typing out URLs can be tedious.

Show transition effects

Un-checking this option will disable any animation in the Sandvox interface. This may be helpful if you have a slower computer. On some computers, even with this option checked, some of the more "demanding" transitions will not be used.

Image Preferences

Please see the "Images" article for more information.

Sandvox provides a small test image but you can drag and drop any other image into the preview area to temporarily set a different test thumbnail.

Sharpen after scaling

Selects how much sharpening should be performed on scaled images.

Preferred Format

Selects the preferred format for images that Sandvox creates.

Quality

If you select "JPEG," use the slider to select the quality of the JPEG images. For more information, please see the "JPEG Compression" article.
