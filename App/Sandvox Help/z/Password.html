Password

When setting up your host, you need to enter the password you use to log into the server. Your Host should provide this for you.

You need to enter your password when Publishing to a Remote Host or Publishing to MobileMe.

If you are connecting to your host by SFTP, you can use a public key instead of a password. Please see "Uploading using SFTP" for information on this.
