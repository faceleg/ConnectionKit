Sandvox Badge

A Sandvox Badge is a simple pagelet to advertise that your site was created with Sandvox. Visitors can click on the badge to go to the Sandvox website.

When a new site is created, the Home Page automatically contains a Sandvox Badge in the Sidebar. We hope you'll want to keep the badge there, but if not it's easy to remove like any other pagelet.

Editing

To edit a Sandvox Badge, you use the Selection Inspector. In addition to those listed below, there are some general pagelet attributes that can be set.

Badge Type

Six badge designs are available.

Anonymous Referral

When a visitor clicks on the Sandvox badge, they are taken to the Sandvox website. Normally, the link they are sent to contains a unique code that allows Karelia to identify the site they came from. If you wish to avoid us knowing this information, check the box to make the link anonymous.

Removing the Sandvox Badge

The Sandvox badge is just a pagelet, which can be removed using any of the following techniques:

  • Click the pagelet to select it, and choose "Delete Pagelet" from the Edit menu.
  • Select the pagelet and press command-option-backspace.
  • Control-click the pagelet and choose "Delete Pagelet" from the contextual menu.
