General Page Attributes

The Page Inspector is split into two halves. The top contains settings that are applicable to all page types; the bottom holds the settings (if any) that are specific to the type of page you are inspecting. This article lists the general settings from the top of the Inspector.

Title

The page's title.

Site Menu

Check the "Include in site menu" box to have a link to the page appear in the Site Menu.

If you fill in the "Menu title" field, that is the title that will appear in the Site Menu. Leave it blank and the page's title will appear instead.

Include in site maps

Choose if the page should appear in site maps. This affects both Sandvox Site Map pages and Google Sitemaps. Not available for the Home Page.

Include in collection index

Uncheck the box to stop the page appearing in its parent collection's index. Not available for the Home Page.

Show sidebar

Hide or show the sidebar on this page.

Checking the "Include pagelets from parent" option will display pagelets from the parent collection's sidebar as well. However if you uncheck the "Show on child pages" option of any such pagelets, they will not be included.

Enable reader comments

Allows readers to leave comments on the page. For this option to work, you must also enable comments for the site as a whole.

Note that for some page types on which leaving comments would be unsuitable (such as Contact Forms) this setting has no effect.

Date/time

Check the box to display the page's date and time. Sandvox should automatically set this correctly for you, but you can also set the date and time yourself.

Thumbnail

Drag an image here to set the page's thumbnail.

Draft

Marks the page as a draft. Not applicable to the Home Page.
