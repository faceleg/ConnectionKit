Metrics Inspector


The Metrics segments is used to adjust Width and Height of an image to the pixel.
Changing the height of an image will also change the width.
You can make an image go back its original size by clicking the "Original Size" button.

With the "constrain proportions" box checked you can adjust the size of the image on the page with the mouse and it will keep your image height-width proportionate.
You can uncheck the "constrain proportions" box if you do not want the object to be height-width proportionate.
