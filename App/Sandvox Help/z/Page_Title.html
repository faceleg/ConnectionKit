Page Title

A page's title is displayed at the top of its main content area:

Page titles also appear in the Site Outline next to each page's icon, and in the Site Menu if applicable.

To set a page's title:

  1. Select the page in the Site Outline.
  2. Open the Page Inspector.
  3. Enter the new page title in the "Page Title" box.

You can also set a page's title by clicking it in the Web View and editing it like any other text.

Sandvox uses a page's title to automatically set its file name.
