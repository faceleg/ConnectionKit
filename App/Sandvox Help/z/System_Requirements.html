System Requirements

To run Sandvox on your computer requires:

  • A PowerPC or Intel Macintosh computer. (Sandvox is a Universal Application).
  • Mac OS X "Tiger" version 10.4.11, Mac OS X 10.5 "Leopard," or Mac OS X 10.6 "Snow Leopard." Sandvox does not run on versions before 10.4.10, including 10.3 (Panther) or 10.2 (Jaguar).
  • Due to massive internal improvements in the operating system, we recommend that you use Leopard or Snow Leopard for larger sites.
  • To use graphical text Sandvox requires a Quartz Extreme-capable Mac. Please see this Apple Support article for details. (Running under the initial release of Mac OS X 10.5 "Leopard" from an Intel machine, this feature will not work properly. We are working with Apple to resolve this issue in a future update to Leopard.)


Sites published by Sandvox are viewable with almost any web browser, although there are two small caveats:

  • Graphical text and PNG images are not properly viewable in Internet Explorer 6 and earlier due to an Internet Explorer bug. To compensate, graphical text is replaced with plain text for visitors using Internet Explorer 6 and earlier.

    Please see "My site looks different when viewed with some web browsers" for further details. Both graphical text and PNG images will appear correctly in Internet Explorer 7 and up.

  • If a site visitor's web browser does not support CSS (all modern web browsers support CSS), only the textual content and basic images from the site will appear. It may be helpful to include a link on your site to the latest version of Internet Explorer to allow visitors to download it so that your page may be viewed as intended.
