Site backups

In the Sandvox Preferences, you can set Sandvox to automatically create a backup of any documents you open.

A backup is an identical copy of the site in a file titled "Backup of" and then the name of the site. Backups are created in the same folder as the original document. If a backup of a document already exists when you open it, Sandvox will delete the old backup.

To set Sandvox to make backups:

  1. Open the Sandvox Preferences by choosing "Preferences" from the Sandvox menu.
  2. Set the "When opening document" popup button to "Create backup."
