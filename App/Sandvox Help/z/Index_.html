Index

An index is a list of all or some of the pages in a Collection. It appears as part of a Collection page like so:

Note that Sandvox also has a Index pagelet.

Index Types

There are five different types of index available to use:

General Text
Page Titles and summaries in a simple list.
Download List
A four column arrangement of page thumbnails, titles, summaries, and dates.
Title List
Just the Page Titles in a simple list.
Photo Grid
Page thumbnails and titles in a grid arrangement.
No Index
You can also elect to have no index at all.

Settings

You can edit the index like any other page in a collection, using the Page Inspector. For information on the available settings, please see the "Collection" article.
